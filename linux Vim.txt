（纯命令文本编辑器）
进入vim：
	vi 文件名
退出vim：
	：w 保存修改
	：w new_filename 保存为指定文件
	：wq 保存并退出
	ZZ 快捷键，保存并退出
	：q！ 不保存并退出
	：wq！保存并退出，文件所有者及root可使用

INSERT（插入模式）：
	i a o
	a：光标所在字符后插入
	A：所在行尾插入
	i：字符前插入
	I：行首插入
	o：光标下插入新行
	O：光标上插入新行
命令模式：
（ESC键），（输入“:”输入编辑模式命令）

定位命令：
	: set nu 设置行号
	: set nonu 取消行号
	gg 到第一行
	G 到最后一行
	nG 到第n行
	: n 到第n行
	$ 移至行尾
	0 移至行首
删除命令：
	x 删除光标所在处字符
	nx 删除所在处后n个字符
	dd 删除所在行，ndd删除n行
	dG 删除所在行到文件末尾
	D 删除所处到行尾
	：n，nd 删除指定范围的行
复制和剪切命令：
	yy 复制当行
	nyy 复制当前行以下n行
	dd 剪切当前行
	ndd 剪切当前行以下n行
	p、P 粘贴载当前光标所在行下、上
替换和取消命令：
	r 取代光标所处字符
	R 从光标处开始替换字符，Esc结束
	u 取消上一步操作
搜索和搜索替换
	/string 搜索指定字符串 （按“n”寻找下一个）
	：set ic 搜索忽略大小写 
	n 搜索指定字符串的下一个出现位置
	：%s/要替换字符串/新替换字符串/g 全文替换指定字符串
		：n1，n2s起始行到终止行
		g 为不询问，c为替换询问
	：n1，n2s/old/new/g 在n1，n2范围内替换指定字符串

特殊功能命令：（转义符为“\”）
：r文件名路径 （导入文件内容至光标处）

:! shell命令 （不退出vim下执行shell命令）

：r ！shell命令 （导入命令执行结果）

：n1，n2s/^/替换为字符/g	//^为行首字符

：n1，n2s/^#//g	//去掉行首的#

：ab 输入字符 替换为字符 （输入字符并空格后自动转换）

编辑vim快捷键：
	：map 编辑键 执行命令 （例：:map ^/ I# <ESC> 快捷键为crtl+/ 执行命令为I#（在行首插入#）然后按下ESC键 ；按键为<>）

保存快捷键或自动替换命令：
	在用户/root目录下创建.vimrc文件，并输入（不用加“：”）
